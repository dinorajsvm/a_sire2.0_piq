Management Dashboard -Quick Guide

Drop Down Option 

Vessel/Site: 

Display all the active vessels based on Logged in company user.

My Vessels: 

Display the Vessel Name based on the mapping done in Management Resource Tab in Vessel Master Screen.

My Backup Vessels: 

Display the Vessel Name based on the mapping done in Management Resource Tab under Backup Vessel in Vessel Master Screen.

Other Vessels: 

Display the Vessel Name based on the mapping done in the Vessel Mapping Screen.

Advanced Search: 

Using the Advanced search option, multi-selection can be possible.
Click on the Search button, the system will display the data in the dashboard based on the selection.

Search Button: Click on the Search button to display the records based on the selection.

Clear Button: Click on the Clear button to clear the selected items.

Summary of selected details will be shown as below while mouse hovering near filter icon.

Quick Guide: 

When click on the “i” icon, it will navigate to the quick guide page in the new tab.

Doughnut Chart:

Date Range:
1) By Default, the date range will be blank, and the system will display Overdue/Open data as on date. 

2) Once the user selects the date range, the system will display the data based on the selection.

3) Status Overdue/ Open applicable only for Doughnut Chart.

Note: By default, while opening the dashboard “Overdue” counts will be shown.

 


Bar Chart: 

By default, the Planned and Overdue count of Internal Audit and Ship Visit Plan will be shown as on date.

  1. Once the user selects the date range, the system will display the data based on the selection.

Ageing (Ageing Data is independent of any Date Filters) 

Forms Involved in this Grid 

  1. Vessel Certificate
  2. Failure Report
  3. Internal Audit
  4. Ship Visit

Condition 

1)Expiry /Due Within 30 Days data will be shown as on date. 

2) Overdue < 30 Days, >0~30 Days, >30 Days data will be shown as on date.

Note: The date range is not applicable for this Grid.
